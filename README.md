# مناسبة (Munasba) 🎉

تطبيق إدارة المناسبات الشامل - أنشئ، نظم، وشارك مناسباتك بسهولة.

## ✨ الميزات

- 🎨 تصميم عصري وأنيق
- 🌙 دعم الوضع الداكن
- 📅 إنشاء مناسبات متعددة الخطوات
- 👥 إدارة قائمة الضيوف
- 💰 تتبع الميزانية
- ✅ قائمة مهام التحضير
- 📍 تحديد الموقع على الخريطة
- 🔗 مشاركة الدعوات عبر QR Code
- 🔔 تذكيرات وإشعارات ذكية

## 🚀 التشغيل

```bash
flutter pub get
flutter run
```

## 📱 المتطلبات

- Flutter >= 3.0.0
- Dart >= 3.0.0
